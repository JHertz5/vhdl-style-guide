Whitespacing Rules (100 - 199)
##############################

entity_specification_100
^^^^^^^^^^^^^^^^^^^^^^^^

This rule checks for a single space after the colon.

**Violation**

.. code-block:: vhdl

   attribute coordinate of comp_1 :component is (0.0, 17.5);
   attribute coordinate of comp_1 :    component is (0.0, 17.5);

**Fix**

.. code-block:: vhdl

   attribute coordinate of comp_1 : component is (0.0, 17.5);
   attribute coordinate of comp_1 : component is (0.0, 17.5);

entity_specification_101
^^^^^^^^^^^^^^^^^^^^^^^^

**Fix**
